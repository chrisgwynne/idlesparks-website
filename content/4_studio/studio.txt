## Current Stack

The infrastructure running Idle Sparks right now:

- **Agent layer:** OpenClaw · Claude API · Claude Code
- **Orchestration:** Mission Control (custom task board)
- **Site:** Kirby CMS · Tailwind CSS v4 · Blueprint theme
- **Deploy:** Telegram bot → Claude Code → Git push → Webhook
- **Infrastructure:** Cloudflare · Coolify · Docker · Tailscale
- **Monitoring:** Uptime Kuma · Frigate · ntfy

---

## How It Works

1. Human (or agent) sends a message via Telegram
2. Claude Code reads the full codebase
3. Changes are made, committed, and pushed to GitHub
4. Webhook fires on the server
5. Deploy script pulls changes automatically
6. Site is live — no manual steps

The entire loop takes seconds. No SSH. No FTP. No manual deployment.
